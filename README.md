Алгоритм трансформации бинарной структуры через строковое представление.

Решение основано на пошаговой декомпозиции целого числа в список битов, с последующей конвертацией в строковый формат. 
Основная логика заключается в поиске паттерна 01 (переход от нуля к единице) и его замене на 10 (переход от единицы к нулю) с помощью метода replace. 
Такой подход позволяет имитировать «сдвиг» веса бита в старший разряд. Для случая, когда в числе присутствует только одна единица (степень двойки), 
реализована отдельная ветка логики, обеспечивающая инкремент разрядности путем добавления нуля в конец строки. Завершающий этап включает 
обратную сборку числа из модифицированной строки через итерацию по разрядам.
